Pipes in Angular

Default Pipes and Custom Pipe

Pipes are used to transform data from one form to another in the .html file.

pipes are concerned with the .html file

Original Value - {{firstName}}
After Pipe - {{firstName | uppercase}}
Original Value - {{myName}}
After Pipe - {{myName | lowercase}}
Original Value - {{titleName}}
After Pipe - {{titleName | titlecase}}
Original Value -
After Pipe -
Original Value - {{currentDate}}
After Pipe - {{currentDate | date}}
Original Value - {{currentDate}}
After Pipe - {{currentDate | date:'dd-MM-yy'}}
Original Value -{{currentDate}}
After Pipe - {{currentDate | date:'dd-MM-yyyy'}}
Original Value - {{currentDate}}
After Pipe - {{currentDate | date:'dd/MMM/yy'}}

we can't print object directly in .html, so we use json pipe -- this is just for developers to check what data is coming from API

Original Value - {{student}}
After Pipe - {{student | json}}

async pipe is used to directly subsribe the observable in .html

Original Value - {{currentTime}}
After Pipe - {{currentTime | async | date:'HH-mm-ss'}}

Custom Pipe here we created a pipe using command -- ng g p pipeName--- In na.pipe.ts , we have transform function , where we put our logic

Original Value - {{student.state}}
After Pipe - {{student.state | na}}
